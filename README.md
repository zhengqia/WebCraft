# WebCraft

WebCraft is a VicroCode-specific development SKILL for building web apps that fit VicroCode's runtime, design baseline, upload flow, publishing behavior, storage paths, iframe environment, monetization rules, and AI integration expectations.

## 中文简介

WebCraft 是 VicroCode 专用开发 SKILL。调用该技能即可从需求梳理、页面开发、Python/SQLite 后端、AI 接入、运行路径、上传发布到计费逻辑，构建符合 VicroCode 基调与运行规则的网页应用。

## English

Use WebCraft when you need to build, repair, adapt, or package a VicroCode-ready HTML/JS/React/Vite/Flask/Python/SQLite application.
